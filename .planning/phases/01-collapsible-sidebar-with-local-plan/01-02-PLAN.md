| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-04 | Denial of Service | refresh-on-open fetch | low | mitigate | Fetch fires only on isOpen→true, never on an interval or while closed (D-03) — bounded request rate |
| T-01-05 | Information Disclosure (XSS) | error/empty/banner copy + quest names in greyed list | low | mitigate | All copy is static literals; dynamic names render as React text children only — no `dangerouslySetInnerHTML` |
| T-01-06 | Tampering | stale-date comparison from localStorage | low | accept | Date mismatch only toggles greyed UI + banner; no security-sensitive action depends on it (ASVS L1) |
</threat_model>
